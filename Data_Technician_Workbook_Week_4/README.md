# ☁️ Data Technician Bootcamp – Week 4  
## Azure Data Fundamentals (DP-900)

---

## 🎯 Overview

Week 4 focused on cloud computing concepts and applying Microsoft Azure services to real-world data scenarios.

Key areas covered:

- Cloud computing models (IaaS, PaaS, SaaS)
- Public, Private, Hybrid & Community cloud
- Data governance and compliance in cloud environments
- Azure data services and architecture design
- Business case solution using Azure

---

## ☁️ Cloud Fundamentals

Explored:

- Benefits of cloud computing (scalability, cost efficiency, remote access)
- Comparison of major providers (AWS, Azure, GCP)
- Serverless computing concepts
- Cloud deployment models

This built foundational knowledge of how modern organisations manage data infrastructure.

---

## 🔐 Data Laws & Compliance

Reviewed:

- GDPR
- Data Protection Act 2018
- Computer Misuse Act
- PCI DSS (for payment processing)

Focused on how compliance applies when storing and processing data in cloud environments.

## 🐾 Azure Business Scenario – "Paws & Whiskers"

Designed an Azure-based solution for a retail business transitioning from spreadsheets to cloud infrastructure.

Key considerations:

- Azure Blob Storage for scalable data storage  
- Azure SQL Database for structured transaction data  
- Azure Machine Learning for customer behaviour insights  
- Azure Data Factory for automation  
- Power BI for dashboard reporting  

Also addressed:

- Data modelling approach  
- Security & encryption  
- Backup & disaster recovery  
- Scalability planning
